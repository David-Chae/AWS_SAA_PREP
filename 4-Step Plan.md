* [ ] 주요 문서/가이드 읽기

  * IAM 소개 (45분)
  * AWS Cloud Quest: Solutions Architect

### **3단계 (3\~8주차): 복습 및 실전 연습**

목표: 시험 도메인별 학습과 문제풀이로 실전 대비

* 주차별 계획: 매주 1\~2개 도메인 학습
* 각 도메인별 학습 절차:

  1. Domain Review 과정 수강 → AWS 서비스와 주제 연계 파악
  2. Domain Practice 과정 수강 → 문제풀이 + 플래시카드로 지식 강화
  3. AWS Builder Labs, AWS SimuLearn 챌린지로 실습
  4. 도메인별 추가 자료 검토
* 선택 활동: **AWS Power Hour (Exam Prep)** 시리즈 수강 (가상 강의)

---

### **4단계 (8\~9주차): 시험 준비도 점검**

목표: 실제 시험과 유사한 환경에서 최종 점검

* [ ] AWS 공식 모의시험(Official Practice Exam) 응시 (3\~3.5시간, 타이머 설정)
* [ ] 점수 리포트 꼼꼼히 분석
* [ ] 틀린 문제 + 해설 문서 다시 확인
* [ ] 필요 시 2단계 학습 자료로 돌아가 보완

📌 **Tip**

## 📅 9-Week AWS Exam Prep Plan

### **Step 1 (Week 1): Get to know the exam**

**Goal:** Understand exam format, topics, and identify knowledge gaps.

* [ ] Review the exam details page
* [ ] Read the exam guide
* [ ] Take the AWS Certified Solutions Architect – Associate Official Practice Question Set to get familiar with exam-style questions

---

### **Step 2 (Weeks 2–3): Refresh AWS knowledge and skills**

**Goal:** Strengthen core AWS knowledge and services.

* [ ] Review key resources:

  * AWS Well-Architected Framework
  * AWS Security Best Practices
  * Amazon EC2, Amazon S3, Amazon VPC, Amazon Route 53, Amazon RDS, Amazon SQS
* [ ] Take digital courses (optional, based on your needs):

  * AWS Technical Essentials (4 hours)
  * AWS Well-Architected Foundations (3 hours)
  * Introduction to Building with AWS Databases (3 hours)
  * Deep Dive with Security: IAM (includes labs, 8 hours)
* [ ] Hands-on & game-based learning:

  * Walkthrough of the AWS Well-Architected Tool (30 minutes)
  * Introduction to IAM (45 minutes)
  * AWS Cloud Quest: Solutions Architect

---

### **Step 3 (Weeks 3–8): Review and practice**

**Goal:** Strengthen knowledge per exam domain through review, practice, and labs.

* Weekly focus: Cover 1–2 domains each week
* For each domain:

  1. Take the Domain Review course → Understand topics and how they map to AWS services
  2. Take the Domain Practice course → Reinforce with exam-style questions and flashcards
  3. Complete AWS Builder Labs and AWS SimuLearn challenges (if available)
  4. Review additional resources listed for that domain
* Optional: Join **AWS Power Hour (Exam Prep)** virtual sessions for alternative review

---

### **Step 4 (Weeks 8–9): Assess exam readiness**

**Goal:** Simulate the exam and close remaining gaps.

* [ ] Take the **Official Practice Exam** (3–3.5 hours, timed mode)
* [ ] Review your score report carefully
* [ ] Revisit correct and incorrect answers with linked AWS documentation
* [ ] Go back to Step 2 resources if additional review is needed

---

📌 **Tips**

* Use weekdays for shorter study sessions (1–2 hours) and weekends for deeper practice and labs.
* Always take the Official Practice Exam at least once before the real exam.

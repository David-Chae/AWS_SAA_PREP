## Introduction to Amazon Textract

Amazon Textract는 스캔된 문서에서 텍스트, 손글씨 또는 데이터를 추출하도록 설계된 서비스입니다.  
→ 텍스트 및 데이터 추출 전문 서비스 강조

As the name suggests, it specializes in text extraction.  
이름에서 알 수 있듯, 텍스트 추출에 특화되어 있습니다.  
→ 서비스 목적 명확화

→ 기술적 기반 강조

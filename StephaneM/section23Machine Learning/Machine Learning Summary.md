In this lecture, we discuss the various machine learning services available on AWS.

### Rekognition
### Rekognition
# → 이미지/영상 관련 ML 서비스

Rekognition enables face detection, labeling, and celebrity recognition.  
Rekognition은 얼굴 감지, 라벨링, 그리고 유명인 인식 기능을 제공합니다.  
→ 이미지 및 얼굴 분석 기능 핵심 요약

### Transcribe and Polly
### Transcribe와 Polly
# → 음성-텍스트 변환 서비스

Transcribe converts audio into text, providing subtitles or transcripts.  
Transcribe는 오디오를 텍스트로 변환하여 자막이나 전사본을 제공합니다.  
→ 음성 → 텍스트 기능

Polly performs the opposite function by converting text into audio.  
Polly는 반대로 텍스트를 오디오로 변환하는 기능을 수행합니다.  
→ 텍스트 → 음성 기능

### Translate and Lex
### Translate와 Lex
# → 언어/대화형 서비스

Translate offers language translation services.  
Translate는 언어 번역 서비스를 제공합니다.  
→ 다국어 번역 기능

Lex allows you to build conversational bots or chatbots.  
Lex는 대화형 봇 또는 챗봇을 구축할 수 있게 해줍니다.  
→ 챗봇 개발 기능

When combined with Connect, you can create a cloud contact center.  
Connect와 결합하면 클라우드 기반 컨택센터를 구축할 수 있습니다.  
→ Lex + Connect 활용 예시

### Comprehend and SageMaker
### Comprehend와 SageMaker
# → 텍스트 분석/머신러닝 플랫폼

Comprehend provides natural language processing capabilities.  
Comprehend는 자연어 처리(NLP) 기능을 제공합니다.  
→ 텍스트 분석 기능

SageMaker is a fully featured machine learning service accessible to developers and data scientists.  
SageMaker는 개발자와 데이터 과학자가 접근할 수 있는 완전한 기능의 머신러닝 서비스입니다.  
→ 모델 개발/훈련/배포 기능

### Kendra, Personalize, and Textract
### Kendra, Personalize, Textract
# → 문서 검색/추천/데이터 추출 서비스

Kendra is an ML-powered document search engine.  
Kendra는 ML 기반 문서 검색 엔진입니다.  
→ 문서 내 정보 검색

Personalize offers real-time personalized recommendations for customers.  
Personalize는 고객 맞춤형 실시간 추천 서비스를 제공합니다.  
→ 추천 시스템

Textract detects and extracts text and data from various documents.  
Textract는 다양한 문서에서 텍스트와 데이터를 감지하고 추출합니다.  
→ 문서 데이터 추출

## Key Takeaways
## 핵심 포인트
# → 시험 및 실무용 요약

AWS offers a variety of machine learning services tailored for different tasks such as image recognition, speech transcription, and natural language processing.  
AWS는 이미지 인식, 음성 전사, 자연어 처리 등 다양한 작업에 맞춘 머신러닝 서비스를 제공합니다.  
→ 서비스별 역할 요약

Rekognition provides face detection, labeling, and celebrity recognition capabilities.  
Rekognition은 얼굴 감지, 라벨링, 유명인 인식 기능을 제공합니다.  
→ 핵심 기능 강조

Transcribe converts audio into text, while Polly converts text into audio.  
Transcribe는 오디오를 텍스트로, Polly는 텍스트를 오디오로 변환합니다.  
→ 음성 ↔ 텍스트 기능 비교

Translate offers language translation services, and Lex enables building conversational bots.  
Translate는 언어 번역 서비스를 제공하며, Lex는 대화형 봇 구축을 가능하게 합니다.  
→ 언어 및 챗봇 서비스 요약

SageMaker is a comprehensive machine learning platform for developers and data scientists.  
SageMaker는 개발자와 데이터 과학자를 위한 종합 머신러닝 플랫폼입니다.  
→ 모델 개발/훈련/배포 강조

Additional services include Kendra for document search, Personalize for real-time recommendations, and Textract for extracting text and data from documents.  
추가 서비스로는 문서 검색용 Kendra, 실시간 추천용 Personalize, 문서 텍스트 및 데이터 추출용 Textract가 있습니다.  
→ 기타 서비스 요약

# Kendra Overview
# Kendra 개요
# → Amazon Kendra 서비스 소개

## Introduction to Amazon Kendra
## Amazon Kendra 소개
# → 서비스 기본 개념

Amazon Kendra is another machine learning service offered by AWS.  
Amazon Kendra는 AWS에서 제공하는 또 다른 머신러닝 서비스입니다.  
→ Kendra가 ML 기반임을 강조

It is a fully-managed document search service powered by machine learning that allows you to extract answers from within a document.  
문서 내부에서 답을 추출할 수 있는 머신러닝 기반 완전 관리형 문서 검색 서비스입니다.  
→ 문서 검색 + 답변 추출 가능

## Supported Document Types
## 지원 문서 유형
# → Kendra가 처리 가능한 파일 형식 설명

The documents that Amazon Kendra can process include text files, PDFs, HTML files, PowerPoint presentations, Microsoft Word documents, FAQs, and many others.  
Amazon Kendra가 처리할 수 있는 문서에는 텍스트 파일, PDF, HTML, 파워포인트, 워드, FAQ 등 다양한 형식이 포함됩니다.  
→ 다중 문서 유형 지원 강조

There are numerous data sources where these documents may reside, some of which are displayed on the screen.  
이 문서들이 존재할 수 있는 데이터 소스는 다양하며, 일부는 화면에 표시됩니다.  
→ 다양한 데이터 소스 지원

## Knowledge Index Creation
## 지식 인덱스 생성
# → 내부 처리 과정

Amazon Kendra indexes these documents and internally builds a knowledge index powered by machine learning.  
Kendra는 문서를 인덱싱하고 머신러닝 기반 지식 인덱스를 내부적으로 구축합니다.  
→ 인덱스 생성과 ML 기반 검색 핵심

## End-User Benefits
## 최종 사용자 혜택
# → 사용자 관점 기능 설명

From an end-user perspective, Amazon Kendra provides natural language search capabilities similar to those of Google.  
최종 사용자 관점에서 Kendra는 구글과 유사한 자연어 검색 기능을 제공합니다.  
→ 자연어 질문 가능

For example, if a user asks, "Hey, where is the IT support desk?" Amazon Kendra can respond with "1st floor."  
예를 들어, 사용자가 "IT 지원 데스크가 어디인가요?"라고 질문하면 Kendra가 "1층입니다."라고 답할 수 있습니다.  
→ 질문-답변 기능 예시

This is possible because Kendra understands from all the resources it has indexed that the IT support desk is located on the first floor, which is quite impressive.  
이는 Kendra가 인덱싱한 모든 자료를 이해하고 IT 지원 데스크가 1층에 있음을 인식하기 때문에 가능한 일입니다.  
→ ML 기반 이해 강조

## Incremental Learning
## 점진적 학습
# → 사용자 피드백 반영

Users can perform normal searches, and Amazon Kendra learns from user interactions and feedback to promote preferred search results.  
사용자가 일반 검색을 수행하면 Kendra는 사용자 상호작용과 피드백을 학습하여 선호 검색 결과를 우선 제공합니다.  
→ 사용자 행동 기반 학습

This process is known as incremental learning.  
이 과정을 점진적 학습(incremental learning)이라고 합니다.  
→ 학습 방식 명칭 강조

## Fine-Tuning Search Results
## 검색 결과 미세 조정
# → 검색 최적화 방법

Additionally, you can fine-tune the search results based on factors such as the importance of data, data freshness, or any custom filters you may have.  
또한 데이터 중요도, 최신성, 사용자 정의 필터 등 다양한 요소를 기반으로 검색 결과를 미세 조정할 수 있습니다.  
→ 맞춤형 검색 가능

시험 관점에서, 문서 검색 서비스를 접하면 Amazon Kendra를 떠올리세요.

## Key Takeaways
## 핵심 포인트
# → 시험/실무용 요약

Amazon Kendra is a fully-managed document search service powered by machine learning.  
Amazon Kendra는 머신러닝 기반 완전 관리형 문서 검색 서비스입니다.  
→ ML 기반 문서 검색 강조

It indexes various document types such as text, PDF, HTML, PowerPoint, Microsoft Word, and FAQs.  
텍스트, PDF, HTML, 파워포인트, 워드, FAQ 등 다양한 문서 유형을 인덱싱합니다.  
→ 다중 문서 지원 강조

Kendra provides natural language search capabilities, enabling users to ask questions and receive precise answers.  
Kendra는 자연어 검색 기능을 제공하여 사용자가 질문하고 정확한 답변을 받을 수 있습니다.  
→ 사용자 질의 응답 기능

The service supports incremental learning by adapting search results based on user interaction and feedback.  
이 서비스는 사용자 상호작용과 피드백을 기반으로 검색 결과를 점진적으로 학습합니다.  
→ 점진적 학습 지원

Search results can be fine-tuned using factors like data importance, freshness, and custom filters.  
검색 결과는 데이터 중요도, 최신성, 사용자 정의 필터 등을 사용해 미세 조정할 수 있습니다.  
→ 검색 최적화 가능

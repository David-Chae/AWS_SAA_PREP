## Introduction to Amazon AppFlow  
## 아마존 앱플로우 소개  
Amazon AppFlow is a fully managed integration service that enables you to transfer data between Software-as-a-Service (SaaS) applications and AWS.  
Amazon AppFlow는 완전 관리형 통합 서비스로, SaaS(Software-as-a-Service) 애플리케이션과 AWS 간 데이터를 전송할 수 있게 해줍니다.  

These integrations are typically difficult to write, but with AppFlow, they become very easy.  
일반적으로 이러한 통합 작업은 작성하기 어렵지만, AppFlow를 사용하면 매우 쉽게 구현할 수 있습니다.  

---

## Supported Data Sources  
## 지원되는 데이터 소스  
Sources of data could include Salesforce, SAP, Zendesk, Slack, and ServiceNow.  
데이터 소스에는 Salesforce, SAP, Zendesk, Slack, ServiceNow 등이 포함될 수 있습니다.  

Salesforce is particularly notable as it may appear in certification exams.  
특히 Salesforce는 인증 시험에서 자주 등장할 수 있으므로 주목할 만합니다.  

---

## Data Destinations  
## 데이터 대상  
You can send data to destinations such as Amazon S3, Amazon Redshift, or even non-AWS targets like Snowflake and Salesforce.  
데이터는 Amazon S3, Amazon Redshift와 같은 AWS 서비스나 Snowflake, Salesforce와 같은 비-AWS 대상에도 전송할 수 있습니다.  

---

## Integration Scheduling and Triggers  
## 통합 예약 및 트리거  
AppFlow integrations can be configured to run on a schedule, in response to specific events, or on demand.  
AppFlow 통합은 예약 실행, 특정 이벤트 발생 시 실행, 또는 필요 시(on-demand) 실행하도록 구성할 수 있습니다.  

---

## Data Transformation and Security  
## 데이터 변환 및 보안  
Within AppFlow, you have data transformation capabilities such as filtering and validation.  
AppFlow 내에서는 필터링과 검증과 같은 데이터 변환 기능을 제공합니다.  

The data is encrypted when transferred over the public internet, or you can transfer it privately using PrivateLink.  
데이터는 공개 인터넷을 통해 전송될 때 암호화되며, PrivateLink를 사용하면 프라이빗하게 전송할 수도 있습니다.  

---

## Benefits of Using AppFlow  
## AppFlow 사용의 장점  
The idea behind AppFlow is that you do not spend time writing the integration yourself.  
AppFlow의 핵심 아이디어는 통합 코드를 직접 작성할 필요가 없다는 것입니다.  

Instead, you leverage these APIs immediately and enjoy the data within your own AWS accounts.  
대신 제공되는 API를 즉시 활용하여 자신의 AWS 계정 내에서 데이터를 사용할 수 있습니다.  

---

## AppFlow Interface Overview  
## AppFlow 인터페이스 개요  
In the AppFlow interface, you select a source from many different options, including the important ones mentioned earlier such as Salesforce.  
AppFlow 인터페이스에서는 Salesforce와 같이 앞서 언급한 중요한 소스를 포함한 다양한 옵션 중에서 소스를 선택합니다.  

Then, you can send this data into various destinations like Redshift or S3.  
그 후, 데이터를 Redshift 또는 S3와 같은 다양한 대상에 전송할 수 있습니다.

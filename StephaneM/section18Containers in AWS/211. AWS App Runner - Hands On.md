# 🟦 **AWS App Runner Hands-On — 쉽게 이해하기**

## 🔧 **App Runner가 뭘 하는 건데?**

App Runner는 **“컨테이너만 있으면 자동으로 서버를 만들어주는 서비스”**야.
서버 세팅, 로드밸런서, 확장 설정 같은 걸 네가 직접 하지 않아도 AWS가 전부 대신 해줘.

다만 **완전히 무료는 아니고**, CPU/메모리 사용량에 따라 비용이 나가.

---

# 🟩 **1. App Runner 서비스 만들기 시작**

서비스를 만들 때 **배포 소스(source)** 를 선택해야 함:

1. **컨테이너 이미지 (Docker Image)**
2. **소스코드(GitHub 등)**

여기서는 *컨테이너 이미지를 직접 선택하는 방식*으로 진행.

---

# 🟨 **2. 어떤 컨테이너를 쓸지 선택**

컨테이너 이미지는 두 군데에서 가져올 수 있음:

* **Amazon ECR(Private)**: 회사 내부용 이미지
* **ECR Public**: 공개된 이미지

이번 예시는 **ECR Public**에서 **HTTPD(Apache 서버)** 이미지를 가져옴.

HTTPD 이미지는 웹페이지 기본 테스트용으로 많이 쓰는 단순 웹 서버야.

> 즉, “심플한 웹 서버를 App Runner로 띄워보자” 이 목적.

---

# 🟧 **3. 배포 방식 선택 (Manual vs Automatic)**

* **Automatic**: 소스코드가 변경되면 App Runner가 자동 재배포
* **Manual**: 사람이 직접 눌러서 배포

이번 예시는 **Manual(수동)**.

---

# 🟥 **4. 서비스 기본 설정**

서비스 이름 지정:

* 예: `DemoHTTP`

CPU/메모리 선택:

* 최소: **1 vCPU, 2GB RAM** → 이 데모엔 충분함

환경변수 넣을 수 있고
컨테이너가 어떤 포트로 서비스 하는지도 알려줘야 하는데,

HTTPD 이미지는 **포트 80**을 사용하므로
App Runner에서도 서비스를 **80번 포트**로 설정.

---

# 🟪 **5. Auto Scaling 설정**

App Runner는 자동으로 트래픽을 보고 서버를 늘리거나 줄여줌.

기본값:

* 최소 1개 ~ 최대 25개 인스턴스
* 한 인스턴스가 동시에 처리할 수 있는 요청: 100개

원하면 직접 숫자를 바꿀 수도 있음.

---

# 🟫 **6. Health Check / 보안 / 네트워크**

### ✔ Health check

서버가 정상 동작하는지 App Runner가 자동 체크
(기본 설정 그대로 둠)

### ✔ Security

컨테이너 내부에서 AWS 서비스에 접근하려면 IAM Role 추가 가능
(예제에서는 패스)

### ✔ Networking

* **VPC 안에 넣을 수 있음** (DB 접속 등)
* 예제에서는 *그냥 Public*로 진행

### ✔ Tracing (X-Ray)

서비스 흐름 분석용
이번에는 끔.

모든 설정을 확인하고 **Create**!

---

# 🟦 **7. App Runner가 내부에서 해주는 일 (Behind the Scenes)**

App Runner는 네가 볼 수 없지만 뒤에서 자동으로:

* 로드밸런서 생성
* 자동 확장 설정
* 컨테이너 배포
* 도메인 주소 생성
* 로그 시스템 연결

**이 모든 걸 완전 자동으로 세팅함.**

→ 이게 App Runner의 핵심 장점.

---

# 🟩 **8. 서비스 완성!**

조금 기다리면 서비스가 완성됨.
App Runner가 제공한 기본 도메인을 클릭하면

```
It works!
```

라는 기본 웹페이지가 뜸.
→ Apache HTTPD 서버가 정상적으로 돌아간다는 의미.

# 🟨 **9. 모니터링**

App Runner는 내부에서:

* 로그
* 이벤트
* 배포 기록
* 스케일링 상태
* 에러
* 메트릭(grafana 느낌)

이런 걸 전부 한 화면에서 확인할 수 있음.

트래픽이 몰리면 App Runner가 자동으로 서버 수를 늘림.

---

# 🟥 **10. 끝낼 때는 한 번에 삭제**

서비스 삭제도 매우 간단:
**delete** 입력 → 바로 삭제됨

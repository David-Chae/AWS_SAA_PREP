## Introduction to AWS Certification Paths

➡ AWS 인증 시험은 기초, 어소시에이트, 프로페셔널, 스페셜티 레벨로 나뉜다.

AWS 인증에 도전하기로 한 것을 축하합니다. 하지만 이것은 여정의 시작일 뿐입니다.  
➡ 자격증 도전은 시작이고, 커리어를 열어가는 첫걸음.  

AWS certifications are organized into foundational, associate, professional, and specialty levels.  
AWS 인증은 기초, 어소시에이트, 프로페셔널, 스페셜티 단계로 구성되어 있습니다.

여러분이 원하는 직무에 따라 추천되는 자격증 경로가 다릅니다.

## Architecture Path  
## 아키텍처 경로  
➡ 클라우드 솔루션 설계자를 목표로 하는 경로.  

If you want to become a solutions architect...  
솔루션 아키텍트가 되고 싶다면...  
➡ 클라우드 인프라 설계·운영 전문가를 의미.  

Recommended certifications:  
추천 자격증:  
- Cloud Practitioner (optional, but recommended)  
  - 클라우드 기초 지식 (선택이지만 추천됨)  
- AI Practitioner Foundational  
  - AI 활용을 원한다면 기초 AI 자격증  
- Solutions Architect Associate  
  - 아키텍처 기본 인증  
- Solutions Architect Professional  
  - 아키텍처 심화 인증  
- Security Specialty  
  - 보안 전문화 자격증 (강력 추천)  

🎮 게임보상: *"Infrastructure Designer"* 칭호 획득!

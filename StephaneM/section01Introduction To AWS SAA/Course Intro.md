## What is AWS?  
## AWS란 무엇인가?  
*(AWS의 개념 설명)*  

AWS stands for Amazon Web Services.  
AWS는 Amazon Web Services의 약자입니다.  

It is a cloud provider that offers servers and services you can use on demand and scale easily.  
AWS는 필요할 때 즉시 사용할 수 있고 손쉽게 확장할 수 있는 서버와 서비스를 제공하는 클라우드 공급자입니다.  
*(온디맨드 및 확장성 강조)* ☁️  

This is a significant difference from traditional on-premises data centers, where you had to physically order and install servers before using them.  
이는 서버를 물리적으로 구매하고 설치해야 하는 전통적인 온프레미스 데이터 센터와의 큰 차이입니다.  
*(클라우드와 기존 데이터 센터 차이)*  

With AWS, you can obtain servers whenever you want, on demand, and scale to as many as you need.  
AWS를 통해 원하는 때에 서버를 확보하고 필요에 따라 확장할 수 있습니다.  
*(유연성 강조)*  

Thanks to AWS, the cloud has been revolutionized. Today, AWS powers some of the biggest websites in the world.  
AWS 덕분에 클라우드는 혁신을 이루었고, 오늘날 세계 최대 규모의 웹사이트들이 AWS 위에서 운영됩니다.  
*(혁신성과 영향력 설명)* 🚀  

You might expect Amazon.com to use AWS since AWS belongs to Amazon.  
AWS가 아마존에 속해 있으니 Amazon.com이 AWS를 사용할 것이라 예상할 수 있습니다.  

However, companies like Netflix do not own a single server; they rent everything from AWS, which allows them to operate at scale.  
그러나 Netflix 같은 기업은 단 한 대의 서버도 소유하지 않고, 모든 것을 AWS에서 임대하여 대규모 운영을 가능하게 합니다.  
*(대기업 사례로 AWS 활용 설명)* 🎬
